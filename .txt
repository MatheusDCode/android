

<div>: Um contêiner genérico para agrupar elementos, frequentemente usado para fins de estilo ou layout.
<header>: Normalmente usado para definir um contêiner para conteúdo introdutório ou links de navegação no topo de uma página ou seção.
<footer>: Usado para definir um contêiner para conteúdo de rodapé, frequentemente contendo informações de direitos autorais ou links relacionados.
<nav>: Usado para definir um contêiner para links de navegação.
<main>: Usado para definir a área de conteúdo principal de um documento.
<article>: Representa uma composição independente que pode ser distribuída ou reutilizada de forma independente.
<aside>: Representa uma seção de uma página que está tangencialmente relacionada ao conteúdo principal.
<section>: Define um agrupamento temático de conteúdo, geralmente com um título.
<hgroup>: Agrupa um título (por exemplo, <h1> - <h6>) com outro conteúdo relacionado, como um parágrafo.
<fieldset>: Agrupa elementos de formulário relacionados, geralmente com uma legenda para um título.
<optgroup>: agrupa opções relacionadas dentro de um elemento <select> (lista suspensa).


---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 Tags de agrupamento (grouping tags):

- Sem valor semântico:

   <div>		-> block
   <span>	-> inline

- Com valor semântico:

  <header>		-> cabeçalho
       <nav>	        -> para menus, links, navegação

 <main>			       -> principal
   <section>		       -> seção
	 <article>	       -> artigo
		 <aside>	       -> periférico ao artigo (ex: escritor / autor)

 <footer>	               -> rodapé
